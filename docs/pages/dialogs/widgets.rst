========
Виджеты
========

Виджеты — это компоненты, из которых строится содержимое окна (``Window``). Все виджеты делятся на текстовые, клавиатурные, медиа-виджеты и поля ввода.

Текстовые виджеты
=================

Отвечают за текстовое содержимое сообщения.

- ``Const``: Статический текст.
- ``Format``: Динамический текст с форматированием (использует ``.format()``) из словаря данных.
- ``Jinja``: Шаблоны Jinja2. Автоматически рендерят текст на основе данных.
- ``Multi``, ``Case``: Для условного отображения текста.

.. code-block:: python

    from maxo.dialogs.widgets.text import Const, Format, Jinja

    Const("Привет!")
    Format("Твой баланс: {balance}")
    Jinja("Привет, {{ user.name }}!")

Клавиатурные виджеты
====================

Используются для создания Inline-кнопок.

* ``Button``: Простая кнопка с callback'ом.
* ``Url``: Кнопка-ссылка.
* ``Row``, ``Column``, ``Group``: Группировка кнопок (в ряд, в колонку).
* ``Select``, ``Radio``, ``Multiselect``: Списки выбора и переключатели.
* ``Next``, ``Back``, ``SwitchTo``: Кнопки навигации между окнами без написания обработчиков.

.. code-block:: python

    from maxo.dialogs.widgets.kbd import Button, Row, Select

    Row(
        Button(Const("Да"), id="yes", on_click=on_click),
        Button(Const("Нет"), id="no", on_click=on_click),
    )

Прочие виджеты
==============

* **DynamicMedia / StaticMedia**: Окно может содержать медиа (фото, видео, документ). Вы можете возвращать URL или ID файла.
* **MessageInput**: Слушает текстовые сообщения от пользователя, находясь в конкретном окне.
