
Полезные заметки
=================

Эта страница содержит набор полезных заметок, связанных с использованием бота.

* При изменении сообщений в ЛС/чате есть ``sender``, а при изменении в канале - нет.
* Если бота не сделать админом в чате/канале, то он не будет получать события из него.
* Нет событий на повышение бота до админа и понижения до юзера.
* При удалении чата/канала боту приходит событие ``user_removed``, будто его кикнули из чата/канала.
* Максимум 12 медиа-аттачментов (фото, видео) в одном сообщении. При этом файловый аттачмент может быть только один.
